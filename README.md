# Ushered

A family health advocacy web app — daily check-ins, medication reminders, pain reports, and AI-powered health insights to help families look out for their elders before a silent condition becomes an emergency.

## Our Story

My grandfather passed away from high blood pressure, which he refused to treat because he thought he was fine. My grandmother had a similar denial, but my family got her treatment. Ushered was built so that every grandparent can have someone looking out for them before a silent condition becomes a life-changing emergency.

## Usage

Open `index.html` in any modern browser. No build step or server required.

## Features

- Guided setup (names, medications, health conditions)
- Elder mode: daily mood check-ins, medication tracking, pain reporting, emergency button
- Family mode: dashboard, alerts, activity timeline, AI health analysis
- Bilingual interface (English + Hindi)
- Data stored locally in the browser
